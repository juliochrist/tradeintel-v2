# TradeIntel

TradeIntel is a premium dark-mode trading journal and AI trading intelligence app built with React, Vite, TypeScript, Tailwind CSS v4, Supabase, TradingView, and a Vercel API route for backend-only AI access control.

## Build Order Covered

1. Project setup with Vite + React + TypeScript
2. Scalable folder structure
3. Tailwind v4 design system using the required color tokens and glass style
4. Supabase schema, Auth, and Row Level Security migration
5. Protected routes with persistent Supabase sessions
6. Journal add/edit/delete/filter UI
7. Dashboard metrics, charts, AI insight, and recent trades
8. Reusable TradingView widget
9. AI prompt builder and service layer
10. Backend AI trial limit enforcement
11. Glass UI, hover scale, skeleton-ready components, and responsive layout
12. Vercel-ready configuration
